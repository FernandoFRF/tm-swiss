# Table Tennis Swiss Tournament

Pequeno app para gerenciar campeonatos de tênis de mesa com sistema suíço.

## Requisitos

- Node.js 18+ (recomendado)

## Como rodar

```bash
npm install
npm start
```

Depois abra no navegador:

- http://localhost:3000

## Estrutura

- `src/core` — regras de negócio (players, torneios, sistema suíço)
- `src/server` — API HTTP + servidor dos arquivos estáticos
- `public` — UI simples em HTML + JS vanilla
